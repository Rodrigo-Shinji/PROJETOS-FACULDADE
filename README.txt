Projeto de criar uma landing page para aula de Programação Web do Professor Valdir Amancio, trabalho com principal função de prática e desenvolvimento de habilidade.
O projeto foi criado com a intenção de montar uma landing page baseada no jogo Tales of Arise sem qualquer motivos comercias ou monetários envolvidos.

html
CSS
